## Electron

Electronに慣れる為に簡単なアプリケーションを作成します。

### 概要

- Mac/linux/WindowsでデスクトップAppを作れる。
- [公式](https://electronjs.org/)
- [ドットインストール](https://dotinstall.com/lessons/basic_electron/36201)

### 用語説明

- index.js
  - main process
  - アプリの状態管理を行う。
- hoge.js
  - rendere process


#### インストール方法

node.jsをインストールする。


```
brew install node
```

electronのインストール

```
//package.jsonを作成する。
npm init

// electronをインストールする。
npm i -D electron@latest  -save-dev
```
